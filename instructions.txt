# Pixel Zoo Setup Instructions

This document explains how to reconstruct this interactive Pixel Zoo web application gallery from scratch, generate the 30 procedural pixel-art sprites, and host it locally in a new environment.

---

## Step 1: Create the Web App Frontend

Create the following files in the project workspace:

1. **`index.html`**: Structures the dashboard page layout with a glassmorphic sidebar panel, search/filter bars, a responsive animal grid, and a pixel-art details modal containing magnification controls and palette emulation shaders.
2. **`styles.css`**: Embeds typography, defines dynamic background blur glow orbs, configures custom themes (Midnight Purple, Cyberpunk Neon, Matrix Green, Classic Dark), and implements strict `image-rendering: pixelated;` rules to render retro sprites with crisp block edges.
3. **`app.js`**: Controls dataset configurations, responsive keyword search (by name, habitat, or diet), filters, global slider scaling, modal trigger loops, canvas filter application, and accessibility focus trapping.

---

## Step 2: Generate the 30 Procedural Sprites

If AI image generation models are unavailable or rate-limited, you can programmatically generate retro 50x50 PNG animal sprites using Python and the `Pillow` library:

1. Create a script called `generate_sprites.py`.
2. Define a list of 30 new animal names along with a 10x10 text grid representing their pixel configurations (e.g. body colors, accents, eyes, appendages).
3. Set the rgb values for color maps (e.g., yellow and black for bees, grey and yellow for wolves).
4. Run the script to write all 30 PNG files:
   ```bash
   python3 generate_sprites.py
   ```

---

## Step 3: Run the Web Server

Start a background Python HTTP server on port 8080 to serve the directory:
```bash
python3 -m http.server 8080
```
Open your browser or Web Preview on port 8080 to view, search, scale, and filter all 45 animals.

---

## Step 4: Package the Files into a ZIP Archive

Run the following command to compress the HTML page, CSS styles, JavaScript logic, generator script, instructions text, and all 45 PNG sprites together:
```bash
python3 -c "import zipfile, glob; zipf = zipfile.ZipFile('pixel_zoo.zip', 'w'); [zipf.write(f) for f in ['index.html', 'styles.css', 'app.js', 'generate_sprites.py', 'instructions.txt'] + glob.glob('*_50x50.png')]; zipf.close()"
```
This produces `pixel_zoo.zip` which can be downloaded directly using:
```bash
dl pixel_zoo.zip
```
